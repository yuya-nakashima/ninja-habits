# AGENTS.md

このリポジトリで作業するエージェント向けのメモです。

## 目的

- `ninja-habits` の実装や設計を進める
- 提案、決定事項、次に試したい案を失わないように残す

## 作業ルール

- 新しい提案や改善案が出たら `docs/proposals.md` に追記する
- 実装前に前提や未確定事項があれば `docs/proposals.md` に残す
- 大きな方針変更をしたら、理由も短く記録する
- まだ決まっていないことは断定せず、候補として書く

## ドキュメント配置

- 提案メモ: `docs/proposals.md`
- 必要に応じて設計メモや要件メモを `docs/` 配下へ追加する

## 記録フォーマット

以下のような形で簡潔に残すこと。

```md
## YYYY-MM-DD

- 提案:
- 理由:
- 次のアクション:
```
