# docs

このフォルダは、提案、検討中の案、設計メモを残すための場所です。

- 提案は `proposals.md` に追記する
- 必要ならトピックごとに別ファイルを追加する

## ファイル

| File | 用途 |
| --- | --- |
| `proposals.md` | 提案、検討メモ |
| `security.md` | アカウント管理とセキュリティ方針 |
| `db-design.md` | RDB 前提のDB設計メモ |
| `api-design.md` | Web版をDB保存で提供するための API 設計メモ |

## v1 の位置づけ

v1 は Web 版（PC・スマホブラウザ）の完成を対象にする。
スコープは Cognito 認証、API、DB 保存を含む、習慣トラッキング、振り返り、Wish List のWeb版完成。
ブラウザ保存は正式な保存先として扱わない。
Push配信、テンプレート、公開、コミュニティなどの Paid 拡張は v2 以降。
Free 利用は v3 のスマホアプリで端末保存として提供する。

詳細はワークスペース横断メモの `../../docs/overview.md` の「バージョンロードマップ」と `../../docs/requirements.md` を参照する。
